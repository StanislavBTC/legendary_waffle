**LEGENDARY_WAFFLE** - Кросплатформенный межсетевой экран для серверного оборудования

## Идеи

За базовый минимум взять проект Sniffnet и достичь его уровня. Исполнение будет в виде TUI с графиками. После полноценной разработки TUI версии, будет выпущена GUI. 

## Цели

Цель, написать мощный кросплатформенный межсетевой экран для серверного и домашнего оборудования, аналог проектам Sniffnet и Wireshark.
